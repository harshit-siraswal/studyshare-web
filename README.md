Explanify

📝 Explanify — A student-friendly college portal for syllabus, resources, notices, chatrooms and study-timers.

🚀 Overview

Explanify is a web-app built to bring together everything a college student needs: syllabus, notes/resources, college notices, group chatrooms, and customizable study timers — all in one place. It offers a clean, modern interface and multiple useful features to simplify how students access and share information, interact with peers, and manage their study schedule.

I built this project to solve a common pain point: students navigating multiple platforms (WhatsApp groups, email, Google Drive, notice-boards, etc.) to find class resources, notices, or coordinate with classmates. Explanify tries to bring all those functionalities under one roof — making collaboration, notice-sharing, resource-access, and study scheduling easier and more organized.

🔧 Features

Syllabus Management — view and manage syllabus items (with sorting / filtering).

Resources Section — store and browse class notes, PDFs, and other learning resources.

College Notices — shows announcements/notices in a “social feed” layout (similar to a Twitter/X-style feed).

Chatrooms — supports multiple chatrooms (e.g. general, placement, course-specific), where students can post questions and others can comment (Reddit-style discussion threads).

Authentication — sign-in support (including “Login with Google”).

Custom Study Timer — users can set their own “focus” and “break” durations (fully customizable).

Bookmarking — bookmark important notes/resources for easy access later.

PDF Viewer for Notes — open shared notes as PDF directly in browser for reading or downloading.
