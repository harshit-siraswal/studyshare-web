Vulnerability Disclosure Policy

StudyShare welcomes responsible security reports. If you believe you have found a vulnerability, report it privately so we can review and fix it quickly.

Contact: security@studyshare.in

How to report

  • Email security@studyshare.in with a clear description of the issue.
  • Include reproduction steps, affected URLs, and any proof-of-concept details that help us validate the finding.
  • Please avoid exposing user data, modifying records, or disrupting service availability.

What we expect

  • Do not publicly disclose the issue before we have had a reasonable chance to respond and fix it.
  • Use the least invasive test possible to demonstrate impact.
  • Only access your own data or data explicitly provided for testing.

Response target

We aim to acknowledge valid reports within 3 business days and will coordinate on remediation progress after that.
